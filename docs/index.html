facilguide-ruby

Ruby gem for Facil.guide multilingual tech guides. Browse 260+ senior-friendly tutorials in 5 languages.

About Facil.guide

This Ruby library provides programmatic access to Facil.guide, a platform offering free simplified technology guides designed for seniors and non-technical users. All content is available in 5 languages: English, Spanish, French, Portuguese, and Italian.

Facil.guide covers 260+ step-by-step tutorials across smartphones, tablets, computers, internet basics, and popular apps. No ads, no accounts required -- just clear, accessible guidance for everyday technology tasks.

# Installation

Ruby 3.0+ | Rails 6+ 
# Install the gem
gem install facilguide

# Or add to Gemfile
gem 'facilguide'

# Quick Start

require 'facilguide'

client = FacilGuide::Client.new

# List guides in French
guides = client.list_guides(lang: 'fr', category: 'smartphone')
guides.each do |g|
  puts "[#{g.lang}] #{g.title}"
end

# Get a specific guide in Italian
guide = client.get_guide('video-call', lang: 'it')
guide.steps.each_with_index do |step, i|
  puts "Step #{i + 1}: #{step.title}"
end

# Check available languages
langs = FacilGuide::LANGUAGES
puts langs.join(', ') # en, es, fr, pt, it

# Features

  • Thread-safe, Rails-compatible, lazy enumeration
  • Guides in 5 languages (EN, ES, FR, PT, IT)
  • Step-by-step tutorials
  • Senior-friendly interface
  • No ads or accounts required

# API Reference

GET client.list_guides(lang:, category:) -> Array<Guide>

Browse guides filtered by language (en, es, fr, pt, it) and topic category.

GET client.get_guide(slug, lang:) -> Guide

Full guide with localized steps, screenshots, and troubleshooting sections.

GET client.categories(lang:) -> Array<Category>

All guide categories with translated names and available guide counts.

METHOD client.search(query, lang:) -> Array<Guide>

Full-text search across all guides in any of the five supported languages.
