# FacilGuide

Multilingual tech guide utilities for Ruby by [Facil Guide](https://facil.guide). Simple technology guides for seniors in 5 languages.

[![Gem Version](https://badge.fury.io/rb/facilguide.svg)](https://rubygems.org/gems/facilguide)

## Install

```
gem install facilguide
```

Or add to your Gemfile:

```ruby
gem 'facilguide'
```

## Usage

```ruby
require 'facilguide'

puts FacilGuide::VERSION
puts FacilGuide::BASE_URL
puts FacilGuide::LANGUAGES  # => ["en", "es", "fr", "pt", "it"]
```

## Supported Languages

- English
- Spanish
- French
- Portuguese
- Italian
